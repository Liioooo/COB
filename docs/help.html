Page

Add page
To add a page you must
  • left-click the + icon in the left sidebar.
  • right-click -> New page in the Main view.
  • go to Edit -> New page in the Menu bar.
  • press CTRL + N.
Select page
To select one page you must
  • left-click it.
  • press Space to select the first created page.
To select multiple pages you must
  • press CTRL + A to select all pages.
  • hold CTRL and left-click all the pages you want to select
  • hold left-click and drag the selection box over the pages you want to select.
Change selection
To change selection you must
  • left-click a new page.
  • use the ARROW KEYS to navigate between the pages.
  • hold left-click and drag the selection box over the pages you want to select.
Delete page
To delete a page you must select a page and
  • left-click the Delete icon in the left sidebar.
  • left-click the Delete icon hovering on the right side above the page.
  • go to Edit -> Delete page in the Menu bar.
  • press DELETE.
Copy page
To copy a page to your clipboard you must select a page and
  • left-click the Copy icon in the left sidebar.
  • left-click the Copy icon hovering on the right side above the page.
  • go to Edit -> Copy page in the Menu bar.
  • press CTRL + C.
Cut page
To cut and copy a page to your clipboard you must select a page and
  • left-click the Cut icon in the left sidebar.
  • left-click the Cut icon hovering on the right side above the page.
  • go to Edit -> Cut page in the Menu bar.
  • press CTRL + X.
Paste page
To paste a page from the clipboard to your Main view you must
  • left-click the Paste icon in the left sidebar.
  • left-click the Paste icon hovering on the right side above the page.
  • go to Edit -> Paste page in the Menu bar.
  • press CTRL + V.
Search
To search a page in the Main view you must
  • left-click the Search icon in the left sidebar.
  • press CTRL F.
    • and enter the name of the page you are looking for.
Move page
To move a page you must select a page and
  • hold left-click and move the cursor.
  • press SHIFT + one of the ARROW KEYS.

View

Zoom in/out
To zoom in/out on the Main view you must
  • left-click the Zoom in/out icon.
  • press CTRL and scroll to zoom in or out.
Move view
To move the view you must
  • use the scroll bars on the bottom and right side.
  • press ALT and use the ARROW KEYS to navigate.
  • press ALT and hold left-click while moving the cursor.

File

Open a file
To open a file you must
  • go to file -> open.
  • press CTRL + O.
Create a file
To create a file you must
  • go to file -> new.
  • press CTRL + SHIFT + N.
Save a file
To save a file you must
  • go to file -> save.
  • press CTRL + S.
Save file as
To save a file with a specific name you must
  • go to file -> save as.
  • press CTRL + SHIFT + S.
Export file
To export a file you must
  • go to file -> export.
  • press CTRL + SHIFT + E.
Import file
To import a file you must
  • go to file -> import.
  • press CTRL + SHIFT + I.

Window

Minimize window
To minimize the window you must
  • go to file -> minimize.
  • click the icon in the top right corner.
Maximize window
To maximize the window you must
  • go to file -> maximize.
  • click the icon in the top right corner.
Unmaximize window
To unmaximize the window you must
  • go to file -> unmaximize.
  • click the icon in the top right corner.
Close window
To close the window you must
  • go to file -> close.
  • click the icon in the top right corner.
  • press ALT + F4.
